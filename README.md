## ⚠️ ¡IMPORTANTE!

Hay que tener en cuenta que el proyecto subido a este repositorio de **GitHub** corresponde a mi entorno **local (htdocs)**, por lo que los datos del archivo de **conexión a la base de datos** **no son los mismos** que los utilizados en el servidor de **InfinityFree**.

Para poder conectar en **producción**, fue necesario usar las **credenciales** (**host, usuario, contraseña y base de datos**) proporcionadas por **InfinityFree**.

Además, existen **pequeñas diferencias** entre el entorno **local** y el entorno **en producción**, principalmente en las **rutas**.  
Por ejemplo, en local se utilizan rutas como:

/initial-d/css/...
mientras que en producción se usan rutas directas como:
/css/...

Estos cambios son mínimos, evidentes y necesarios para el correcto funcionamiento en el servidor.
